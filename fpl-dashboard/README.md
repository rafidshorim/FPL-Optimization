# FPL Scout — Fantasy Premier League Dashboard

A production-quality FPL decision-support dashboard. Uses the official public FPL API — no keys required.

**Features:** Squad overview · Lineup optimizer · Transfer engine · GW planner · Player detail drawer · Demo mode

## Quick Start

```bash
cd fpl-dashboard
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000). Enter your FPL team ID (from `fantasy.premierleague.com/entry/{ID}/`) or click **Explore with demo data**.
